## рҹӣ пёҸ the stack (no glassmorphism, no gradient soup, just engineering)

<div align="center">

| Layer | Tech |
|---|---|
| **Frontend** | React 18 В· Vite В· Zustand В· TanStack Query В· Recharts В· Vanilla CSS design tokens |
| **Backend** | Node.js В· Express.js В· Mongoose (MongoDB Atlas) В· Redis В· Passport.js |
| **AI Layer** | LangChain JS + Google Generative AI (`gemini-2.0-flash`) |
| **Mail** | Nodemailer / Brevo SMTP |
| **Auth** | JWT (15-min access / 7-day refresh) В· bcrypt (12 rounds) В· Redis blacklist on logout |
| **Testing** | Jest |

</div>

### рҹ”җ security, taken seriously (not just a Helmet import for show)

- Dual-token JWT system вҖ” short-lived access tokens, HttpOnly refresh cookies
- Redis-backed token blacklist вҖ” logout actually logs you out, everywhere, instantly
- Helmet + CORS + rate limiting вҖ” 100 req/15min general, 5 req/15min on auth routes (credential-stuffing can go cry somewhere else)
- bcrypt @ 12 rounds вҖ” no plaintext passwords, ever

## рҹҡҖ run it yourself

```bash
# clone the crime scene
git clone https://github.com/Notanormaldev/Odoo_EcoSphere.git
cd Odoo_EcoSphere

# environment setup
cp backend/.env.example backend/.env
# fill in: MONGO_URI, JWT_SECRET, GOOGLE_GEMINI_API, REDIS_HOST, REDIS_PORT

# install everything, both sides
npm run install-all

# seed some demo ESG chaos
cd backend && npm run seed

# fire it up (frontend + backend together)
cd ..
node dev.js
```

Open `http://localhost:5173` and pretend you're the Chief ESG Strategist. (Spoiler: the seeded employee literally has that title.)
